# О проекте

Это локальный тренажёр печати: выводит строку для набора, принимает **реальный ввод** и считает:

- `WPM (gross)` — скорость по количеству введённых символов (1 слово = 5 символов)
- `Точность` — сходство введённого текста с эталоном
- `WPM (net)` — скорость с учётом точности

## Основные файлы

- `run_project.py` — точка входа (по умолчанию запускает локальную практику)
- `typing_practice.py` — логика локальной практики
- `doctor.py` — диагностика окружения
- `config.py` — базовый текст (используется, если не задан `--prompt/--prompt-file`)
